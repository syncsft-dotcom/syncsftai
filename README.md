# SYNCSFTAI

SYNCSFTAI is an ownerless, 0% tax BEP-20 token deployed on the BNB Smart Chain with a fixed supply of 10,000,000,000 tokens. The contract includes no mint, burn, blacklist, pause, or admin functions, ensuring a secure, transparent, and decentralized structure with minimal attack vectors.

## Token Information

- **Name:** SYNCSFTAI  
- **Symbol:** SYNCSFTAI  
- **Decimals:** 18  
- **Total Supply:** 10,000,000,000  
- **Network:** BNB Smart Chain (BSC)  
- **Contract Address:** `0xee8C50fca9e57ca21bE64a967f914fA330e9a594`

## Key Features

- 0% tax on buys and sells  
- No owner or privileged roles  
- No mint/burn/pause/blacklist functions  
- Rug-proof design  
- Liquidity locked for 1 year  

## Smart Contract

The token contract was written in Solidity and deployed to BSC.  
The source code has been **verified on BscScan**.

## Liquidity

Liquidity is locked for 1 year to protect holders and prevent rug pulls.
